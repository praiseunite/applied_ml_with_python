## 🚀 Getting Started

### Prerequisites

- **Python 3.10+** installed ([Download](https://www.python.org/downloads/))
- **Git** installed ([Download](https://git-scm.com/downloads))
- **Jupyter Notebook** or **VS Code** with Jupyter extension
- Basic knowledge of Python, pandas, numpy, and matplotlib

### Quick Setup

```bash
# 1. Clone the repository
git clone https://github.com/your-username/Applied_ML_with_Python.git
cd Applied_ML_with_Python

# 2. Create a virtual environment
python -m venv venv
source venv/bin/activate        # Linux/Mac
venv\Scripts\activate           # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Launch Jupyter
jupyter notebook
```

> 📖 For detailed setup instructions, see the [Setup Guide](SETUP_GUIDE.md).
